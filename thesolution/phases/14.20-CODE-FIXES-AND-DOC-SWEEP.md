## Section 1: Code Fixes — CRITICAL

### Fix 1 — JWT `is_system_admin` Claim Missing (Endpoints Non-Functional)

**Problem:** `AdminController.CreateOrganization` and `CreateOwner` check `User.HasClaim("is_system_admin", "true")` but `AuthService.GenerateJwtToken` never emits that claim. Both endpoints permanently return 403 for all users including system admin.

**Files:**
- `api/src/FedProspector.Infrastructure/Services/AuthService.cs` (GenerateJwtToken, ~line 610)
- `api/src/FedProspector.Api/Controllers/AdminController.cs:73-91`

**Fix:** Add `new Claim("is_system_admin", user.IsAdmin == "Y" ? "true" : "false")` to the claims list in `GenerateJwtToken`.

- [x] COMPLETE

---

### Fix 2 — ForcePasswordChange Never Enforced

**Problem:** No middleware blocks API calls when `force_password_change = "Y"`. `AdminService.ResetPasswordAsync` doesn't even set the flag. `OrganizationService.CreateOwnerAsync` sets it but nothing enforces it.

**Files:**
- `api/src/FedProspector.Api/Middleware/ForcePasswordChangeMiddleware.cs` — **CREATE**
- `api/src/FedProspector.Api/Program.cs` — register middleware
- `api/src/FedProspector.Infrastructure/Services/AdminService.cs` (~line 159)

**Fix:**
1. Create middleware that checks `force_password_change` claim/DB field; allows only `POST /auth/change-password` and `POST /auth/logout`; returns 403 otherwise
2. Set `user.ForcePasswordChange = "Y"` in `ResetPasswordAsync`
3. Add `force_password_change` to JWT claims so middleware doesn't need DB lookup per request
4. `AuthService.ChangePasswordAsync` must set `ForcePasswordChange = "N"` after successful change

- [x] COMPLETE

---

### Fix 3 — ExportCsvAsync Cross-Tenant Data Leak

**Problem:** `IOpportunityService.ExportCsvAsync(OpportunitySearchRequest request)` has no `organizationId` parameter. Unlike `SearchAsync` and `GetDetailAsync`, the export bypasses org isolation.

**Files:**
- `api/src/FedProspector.Core/Interfaces/IOpportunityService.cs:11`
- `api/src/FedProspector.Infrastructure/Services/OpportunityService.cs` (ExportCsvAsync)
- `api/src/FedProspector.Api/Controllers/OpportunitiesController.cs` (ExportCsv)

**Fix:** Add `int organizationId` parameter to signature; thread through query; resolve from claims in controller.

- [x] COMPLETE
