# Phase 20.1: Tech Stack Upgrade

**Status**: NOT STARTED
**Dependencies**: Phase 20 (UI Foundation)
**Deliverable**: Node.js 22 LTS + Vite 7 + ESLint 10, all deps at latest compatible versions

---

## Overview

Upgrade Node.js from 20.16 to 22.x LTS, then cascade upgrades to Vite 7, ESLint 10, and related tooling. All application dependencies (React, MUI, TanStack Query, etc.) are already at latest and require no changes.

## Why

- Node 20 EOL: April 2026 (less than a year away)
- Node 22 LTS: supported through April 2027
- Vite 7 (Rolldown-based bundler) requires Node 20.19+
- ESLint 10 requires Node 20.18+
- No breaking changes expected -- all app code is compatible

## Prerequisite (Manual)

User must install Node.js 22.x LTS before this phase can execute:

1. Download from https://nodejs.org (Windows MSI installer, LTS tab)
2. Run installer (overwrites Node 20, global packages in `%APPDATA%\npm` survive)
3. Close and reopen terminal
4. Verify: `node -v` shows v22.x

## Tasks (Automated)

All tasks below are executed by the agent after Node 22 is installed.

- [ ] Confirm `node -v` returns v22.x

## Rollback

If anything breaks after upgrading:
1. Reinstall Node 20.x LTS from nodejs.org
2. `cd ui && rm -rf node_modules && npm install`
3. Everything reverts to the working Vite 6 / ESLint 9 state (package.json still has the old version ranges until committed)

## Risk Assessment

**Low risk.** All application code (React components, API client, hooks, types) is version-agnostic. Only build tooling changes. The upgrade is fully reversible.
