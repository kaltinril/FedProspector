# Phase 20.1: Tech Stack Upgrade

**Status**: COMPLETE
**Dependencies**: Phase 20 (UI Foundation)
**Deliverable**: Node.js 22 LTS + Vite 7, all deps at latest compatible versions. ESLint 10 deferred (plugin compatibility).

---

## Overview

Upgrade Node.js from 20.16 to 22.x LTS, then cascade upgrades to Vite 7 and related tooling. ESLint 10 deferred because eslint-plugin-jsx-a11y does not yet support it. All application dependencies (React, MUI, TanStack Query, etc.) are already at latest and require no changes.

## Why

- Node 20 EOL: April 2026 (less than a year away)
- Node 22 LTS: supported through April 2027
- Vite 7 (Rolldown-based bundler) requires Node 20.19+
- ESLint 10 requires Node 20.18+
- No breaking changes expected -- all app code is compatible

User installed Node.js 22.22.1 via NVM for Windows 1.2.2:

1. Upgraded NVM for Windows from 1.1.9 to 1.2.2

3. Symlink path changed from `C:\Program Files\nodejs` to `C:\nvm4w\nodejs`
4. Verified: `node -v` shows v22.22.1

## Rollback

If anything breaks after upgrading:
1. `nvm use 20.16.0` (or reinstall Node 20.x LTS)
2. `cd ui && rm -rf node_modules && npm install`
3. Everything reverts to the working Vite 6 / ESLint 9 state (package.json still has the old version ranges until committed)

## Risk Assessment

**Low risk.** All application code (React components, API client, hooks, types) is version-agnostic. Only build tooling changes. The upgrade is fully reversible.
